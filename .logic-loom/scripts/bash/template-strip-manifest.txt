# checkout and needs no strip entry — only its tracked `.gitkeep` survives.

#                         asserted to carry NO harness-dev markers (not absent)
#   warn: <path-glob>  -> NOT stripped; leak-guard emits a WARNING (non-fatal).
#                         Use for deferred decisions so nothing silently ships.
# Globs are matched against `git ls-files` output (tracked paths only).
