# sdd-orchestrator

Multi-agent swarm orchestration plugin.

## Commands
| Command | Purpose |
|---------|---------|
| `/swarm` | Custom multi-agent swarm with domain detection |
| `/research` | Deep multi-pass research |
| `/research-team` | 3 parallel researchers + synthesizer |
| `/build-team` | Architect → implementor → reviewer |
| `/review-team` | Security + quality + performance reviewers |
| `/fullstack-team` | Frontend + backend + database parallel |

## Agents: swarm-coordinator, team-synthesizer, task-orchestrator, workflow-coordinator
## Skills: multi-skill-workflow, full-stack-feature, migration-workflow, deep-research, team-orchestration
