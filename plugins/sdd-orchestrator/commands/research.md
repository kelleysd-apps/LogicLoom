# /research Command

Activate the deep-research skill for comprehensive, validated research.

## Usage
```
/research "topic or question to research"
```

## Output
Research directory at `.docs/research/YYYYMMDD-HHMMSS/` with:
- Pass 1: Exploratory findings
- Pass 2: Validation with cross-referencing
- Pass 3: Domain agent synthesis
- Final: Confidence-rated report
