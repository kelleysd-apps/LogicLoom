# /fullstack-team Command

## Team Composition
```
Phase 1 (sequential):
  database-specialist: Design schema and migrations

Phase 2 (parallel):
  backend-architect: Build API and business logic
  frontend-specialist: Build UI components

Phase 3 (sequential):
  full-stack-developer: Wire frontend ↔ backend integration

Phase 4 (parallel):
  testing-specialist: E2E tests
  security-specialist: Auth/security review
```

## Budget Allocation
- Database: 15%
- Backend: 25%
- Frontend: 25%
- Integration: 15%
- Testing + Security: 20%
- Default total: $25.00

## Usage
```
/fullstack-team "Build user profile management with avatar upload"
/fullstack-team "Add payment processing with Stripe" --budget 30.00
```
