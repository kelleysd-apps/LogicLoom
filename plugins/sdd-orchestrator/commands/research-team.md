# /research-team Command

## Team Composition
- **3× Researchers** (parallel): Each explores the topic from a different angle
- **1× Synthesizer** (sequential): Merges findings into unified assessment

## Execution
```
Phase 1 (parallel):
  researcher-1: Primary source research (official docs, specs)
  researcher-2: Community perspective (blogs, forums, real-world usage)
  researcher-3: Comparative analysis (alternatives, benchmarks)

Phase 2 (sequential):
  synthesizer: Cross-reference, resolve conflicts, produce report
```

## Budget Allocation
- Each researcher: 20% of total budget
- Synthesizer: 40% of total budget
- Default total: $10.00

## Usage
```
/research-team "Evaluate GraphQL vs REST for our API layer"
/research-team "Compare React, Vue, and Svelte for our frontend" --budget 15.00
```
