# /review-team Command

## Team Composition
```
Phase 1 (parallel):
  security-specialist: Security vulnerability analysis
  testing-specialist: Code quality and test coverage
  performance-engineer: Performance analysis and optimization

Phase 2 (sequential):
  synthesizer: Merge review findings, prioritize issues
```

## Budget Allocation
- Each reviewer: 25% of total budget
- Synthesizer: 25% of total budget
- Default total: $10.00

## Usage
```
/review-team "Review the authentication module before merge"
/review-team "Full security and performance audit of the API layer"
```
