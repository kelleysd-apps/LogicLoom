# loom-maintenance Plugin

Framework maintenance and lifecycle management.

## Commands
- `/update-framework` — Check for and apply upstream framework updates
- `/initialize-project` — Post-PRD project initialization

## Skills
- `framework-updater` — Upstream sync with cascade impact analysis
- `mcp-server-setup` — MCP server discovery and configuration via Docker Toolkit
- `project-initialization` — Constitution customization and agent training

## Agents
- `framework-sync-agent` — Automated framework update management
