# sdd-specification

SDD specification lifecycle plugin providing the unified `/specification` command and legacy `/specify`, `/plan`, `/tasks` commands.

## Commands
| Command | Purpose |
|---------|---------|
| `/specification` | Unified spec → plan → tasks in one workflow |
| `/specify` | Create feature specification (legacy) |
| `/plan` | Generate implementation plan (legacy) |
| `/tasks` | Generate task list (legacy) |

## Skills: sdd-specification, sdd-planning, sdd-tasks, unified-specification
## Agents: (none — specification lifecycle powered via skills)
